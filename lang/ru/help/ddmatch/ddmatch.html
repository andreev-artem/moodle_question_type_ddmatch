Вопрос на соответствие (с перетаскиванием)

Испытуемому будет показано несколько мини-вопросов и перемешаных ответов. Для каждого мини-вопроса существует только один правильный ответ.

Испытуемый должен выбрать ответ, соответствующий каждому мини-вопросу.

Каждый правильный ответ на мини-вопрос вносит равный вклад в итоговую оценку за весь вопрос.
